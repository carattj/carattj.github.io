Evolution Update Assistant

  • Jacopo Caratti
  • Software Development
  • 2020

With more than 6'000 employees and 600'000 customers, Zucchetti Group is one of the most important Italian companies in the IT sector in Europe.

This project wants to facilitate the grouping of plugins within packages, to be able to download and update them automatically.

I followed the entire development of the project, a single-page web application composed by a fronted and a backend. For the fronted, I used the javascript framework Vue.js; whereas to implement the backend, I used ASP.NET Core.

The main goal of the project is to avoid errors by automating processes. It can be used as a launch pad for future projects.
