Description

With more than 6'000 employees and 600'000 customers, Zucchetti Group is one of the most important Italian companies in the IT sector in Europe.

Evolution Update Assistant is a project designed to facilitate the grouping of plugins within packages, to be able to download and update them automatically.

I followed the entire development of the project, a single-page web application. For the fronted I used the javascript framework Vue.js, whereas to implement the backend, I used ASP.NET Core.

The main goal of the project is to avoid errors by automating processes. It can be used as a launch pad for future projects.
